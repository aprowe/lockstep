Lockstep — User Guide

Chop a video into clips, snap each one to a BPM grid, time-stretch the motion inside so the beats land where they should, then export each clip as its own file.

01

Install

Grab the latest build for your OS from the Releases page. FFmpeg is bundled — there's nothing else to install.

  • Windows — .msi or .exe installer
  • macOS — in development
  • Linux — .AppImage or .deb
Note On macOS, the first launch may need a right-click → Open to bypass Gatekeeper, since the build isn't notarized yet.
02

The interface

The window is split into a few main areas. You can drag the dividers to resize, and rearrange the side panels to taste.

  • Folder sidebar (left) — every video in your current folder, click to switch
  • Video player (top) — preview, scrub, play/pause
  • Timeline (bottom) — scrubber, beat grid, markers, region overlays
  • Side panels (right) — region list, marker table, region info / BPM controls
03

Load a video

A few ways to open something:

  1. Load Video button — click the orange Load Video button in the middle of the empty player
  2. File menu — File → Open Video… or File → Open Folder…
  3. Files sidebar — once a folder is open, switch to the Files tab in the left sidebar and click any video to load it
04

Set the BPM

The beat grid drives everything. BPM is set per clip — pick a clip in the sidebar (or create one — see Regions) and type the BPM into the Clip Info panel.

05

Drop markers

A marker is a moment in the source video that you want to land on a specific beat. Jumps, landings, glances, hits — anywhere motion should sync to rhythm.

  1. Scrub or play to the moment you care about
  2. Press M or use the toolbar mark button to drop a marker at the playhead
  3. Or double-click anywhere on the timeline to drop one right at that point
  4. Repeat for every key moment in the clip
Tip You don't need to be perfect. Land markers near the moment you care about; you can nudge them later by dragging or by stepping frame-by-frame.
06

Align to the beat

Each marker has two handles: where it is in the source video, and where it should land on the beat grid. Drag the beat-side handle onto the bar or beat where the moment should hit.

The exact playback rate for each segment shows up on the Speed row at the bottom of the timeline (e.g. 0.87×, 1.08×) — that's the source of truth for what's being stretched or compressed.

Snapping

Marker handles snap to the beat grid.

07

Regions

A region is a named sub-clip inside your video — its own in/out points, BPM, and lock behavior. Use regions when one source video has multiple sections you want to treat independently.

Creating a region

Add a region from the Clips sidebar. By default a new region snaps to the gap between the two detected scenes around the playhead. To change its bounds, drag its edges directly on the timeline — that's the quickest way to fine-tune the in and out. Then set its BPM in the Clip Info panel.

Lock mode

Each region has a lock setting that controls what stays fixed when you resize it:

  • Lock BPM — resizing changes how many beats fit in the region
  • Lock beats — resizing changes the BPM to keep the same beat count
08

Export

Open the export dialog from File → Export.

What you can do here

  • Single export — render the whole video, or just one region
  • Batch export — render every region at once into a chosen folder
Note Export runs in the background. Progress shows in the top-right of the window; you can keep editing while it works.
10

Keyboard shortcuts

Lockstep ships its own shortcut sheet — open it from the View menu to see the current bindings, grouped by Playback, Markers, and Regions.
