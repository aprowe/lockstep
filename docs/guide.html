Lockstep — User Guide

Warp any video to the beat of any music. Drop markers on the moments you want to hit, line them up to a beat grid, and Lockstep time-stretches everything in between so your cuts land on the beat.

01

Install

Grab the latest build for your OS from the Releases page. FFmpeg is bundled — there's nothing else to install.

  • Windows — .msi or .exe installer
  • macOS — .dmg
  • Linux — .AppImage or .deb
Note On macOS, the first launch may need a right-click → Open to bypass Gatekeeper, since the build isn't notarized yet.
02

The interface

The window is split into a few main areas. You can drag the dividers to resize, and rearrange the side panels to taste.

  • Folder sidebar (left) — every video in your current folder, click to switch
  • Video player (top) — preview, scrub, play/pause
  • Timeline (bottom) — scrubber, beat grid, markers, region overlays
  • Side panels (right) — region list, marker table, region info / BPM controls
03

Load a video

Three ways to open something:

  1. Drag and drop — drop a video file, a folder, or a .json project file onto the window
  2. File menu — File → Open Video… or File → Open Folder…
  3. Folder sidebar — once a folder is open, click any video to switch to it
Tip Opening a folder lets you batch-process: load it once, then walk through every clip and export them all together at the end.
04

Set the BPM

The beat grid drives everything. Set the BPM first so you have something to align against.

Tap tempo

Press T in time with the music — Lockstep estimates the BPM from your taps. Four or more taps gives a stable reading.

Type a BPM directly

If you already know the BPM (e.g. from your DAW), enter it in the region info panel.

05

Drop markers

A marker is a moment in the source video that you want to land on a specific beat. Jumps, landings, glances, hits — anywhere motion should sync to rhythm.

  1. Scrub or play to the moment you care about
  2. Press M (or use the toolbar mark button) to drop a marker at the playhead
  3. Repeat for every key moment in the clip
Tip You don't need to be perfect. Land markers near the moment you care about; you can nudge them later by dragging or by stepping frame-by-frame.
06

Align to the beat

Each marker has two handles: where it is in the source video, and where it should land on the beat grid. Drag the beat-side handle onto the bar or beat where the moment should hit.

The visual connector between the source and beat positions tells you at a glance whether a segment will be sped up (slanted forward) or slowed down (slanted back).

Snapping

Beat-side handles snap to the beat grid by default. Hold a modifier (configurable in Settings) to drag freely between beats.

07

Regions

A region is a named sub-clip inside your video — its own in/out points, BPM, and lock behavior. Use regions when one source video has multiple sections you want to treat independently.

Creating a region

  1. Set in/out points with I and O
  2. Right-click the timeline (or use the region sidebar) to create a region from the current in/out
  3. Name it, set its BPM

Lock mode

Each region has a lock setting that controls what stays fixed when you resize it:

  • Lock BPM — resizing changes how many beats fit in the region
  • Lock beats — resizing changes the BPM to keep the same beat count
08

Export

Open the export dialog from File → Export.

What you can do here

  • Single export — render the whole video, or just one region
  • Batch export — render every region at once into a chosen folder
  • Diagnostic mode — overlays the beat grid on the output so you can verify alignment frame-by-frame
Note Export runs in the background. You'll see progress at the bottom of the window; you can keep editing while it works.
09

Saving your work

Lockstep saves your markers, regions, and BPM settings two ways:

Automatic (per file)

Every change is saved to Lockstep's app data folder, keyed by a content hash of the video. Rename or move the file and it'll still find your work — as long as the file's bytes haven't changed.

Sidecar JSON

Use File → Save Project to write a .json file next to the video. Drop that .json back onto Lockstep later (on any machine) and the sibling video opens with everything restored.

Troubleshooting

Video won't load

Lockstep supports anything FFmpeg can read — which is most things. If a file refuses to open, check the log file (Help → Open Log Folder) for an FFmpeg error.

Export sounds chipmunked or muddy

Time-stretching audio outside ~0.5×–2.0× starts to artifact. If a region is doing extreme stretching, consider splitting it into smaller regions with intermediate beats.

Tap tempo is off

Tap on the downbeat consistently and give it at least 4–8 taps. If a tap feels wrong, just keep going — the latest taps weight the estimate more than older ones.

I lost my work

Open the same video again — Lockstep restores per-file state automatically. If the file's bytes have changed (re-encoded, edited externally), the hash won't match and you'll get a fresh slate. Use sidecar JSONs for important projects.
